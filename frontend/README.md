Minimal React frontend for chat microservices backend.

## Features
- User registration
- Login (JWT)
- Feed viewing (with JWT)
- Post creation (with JWT)

## Usage
1. Run `npm install` in this directory if you haven't already.
2. Run `npm start` to start the frontend at http://localhost:3000
3. Make sure your backend API Gateway is running at http://localhost:8080
4. Use the UI to register, login, view feed, and create posts.

## Project structure
- `src/App.js` - Main app logic and UI
- `src/index.js` - Entry point
- `src/App.css` and `src/index.css` - Minimal styling

All boilerplate/test files are removed or empty for a clean minimal setup.
